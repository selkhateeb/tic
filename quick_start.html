Special Urls¶

We have few urls to help you develop fast

Url ending with ‘_ep/’¶

Example:

http://localhost:8080/example/client/component_ep/

‘ep’ stands for EntryPoint. Runs the specified entrypoint. Useful for running quick scripts or render components a browser.

This will load the file ‘example/client/component_ep.js’ along with its dependancies in an HTML page.
